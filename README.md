### Requirements
    
    - PHP >= 5.4.0
    - ext-curl: *
    - ext-json: *
    - ext-mbstring: *
    
### Composer

You can install the package via [Composer](http://getcomposer.org/). Add this to your `composer.json`:

    {
      "require": {
        ...
        "arcanedev/head": "dev-master"
        ...
      }
    }
    
Then install via:

    composer.phar install
